#### 페이지 설명

1. user 의 아이디와 비밀번호 값이 같지 않을 때는 일치하지 않는다는 안내창이 나옴

2. 값이 일치한다면 welcome 이라는 페이지로 넘어가게끔 코드 구현

3. 아이디와 비밀번호가 정해진 양식과 다르게 입력된다면 에러 메세지를 띄워줌
